## How to run the aplication :arrow_forward:

In terminal, clone the project: 

```
git clone https://github.com/jpadedg/portfolioReact.git
```

```
cd portfolioReact
```

For open the Visual Code Studio:

```
code .
```

In VS code terminal:

```
npm i
```

```
npm start
```
